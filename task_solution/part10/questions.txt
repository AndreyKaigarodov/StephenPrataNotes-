1. Что такое класс ?:
-Класс - пользовательский тип данных, который может содержать методы и поля для работы с данными
поля и методы могут быть приватными и публичными. К приватным полям и методам можно обратитсья только из
класса или friend функции

2. Каким образом класс обеспечивает абстракцию, инкапсуляцию и сокрытие данных ?:
-Абстракция достигается за счет выделения в сущности только необходимых параметров, которые
полно и цельно описывают систему. Инкуапсуляция и сокрытие достигается с помощью модификаторов доступа
и дружественных функций

3. Каково отношение между объектом и классом?
Объект - экземпляр класса, с ним можно работать с помощью методов и полей, описанных в классе.
сам объект может предоставлять только открытые интерфейсы для манипулирования даннымы.

4. Чем отличаются функции-члены класса от данных-членов класса помимо того, что они - функции?
функция-член - содержит указатель на объект ?
данные-члена класса
